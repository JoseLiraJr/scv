SCV - Sistema de Controle de Vendas
----------------------------------------------
O Sistema foi desenvolvido com fins didáticos. 

Foi utilizado a linguagem de programação Object Pascal, a IDE Embarcadero® Delphi 10.3 Version 26.0.36039.7899, FireDAC para o acesso aos dados, SGBD SQLite e o Gerador de Relatórios Fortes Report.

Foi utilizado como padrão a arquitetura MVC (Parcial), pois não foi implementada a camada de negócios (Service).

O Sistema foi feito utilizando herança de telas e o paradigma de programação Orientado a Objetos. 

Observação I: Para reiniciar o banco de dados do projeto é só excluir o arquivo dados.db da pasta database e executar novamente o aplicação (scv.exe) que o banco será criado novamente.

Observação II: O projeto é composto dos cadastros Cliente, Fornecedor e Produtos, da tela de Vendas e Pesquisa de Vendas e dos relátorios gerenciais de Clientes, Fornecedores, Produtos e Vendas.

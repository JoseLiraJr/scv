SCV - Sistema de Controle de Vendas
----------------------------------------------
O Sistema foi desenvolvido com fins didáticos e como parte de processo seletivo da empresa Cozimax. 

Foi utilizado a linguagem de programação Object Pascal, a IDE Embarcadero® Delphi 10.3 Version 26.0.36039.7899, FireDAC para o acesso aos dados e o SGBD SQLite.

Foi utilizado como padrão a arquitetura MVC (Parcial), pois não foi implementada a camada de negócios (Service).

O Sistema foi feito utilizando herança de telas e o paradigma de programação Orientado a Objetos. 

Observação I: Para reiniciar o banco de dados do projeto é só excluir o arquivo dados.db da pasta dados e executar novamente o aplicação (scv.exe) que o banco será criado novamente.

Observação II: Não foi usado o SGDB MicroSoft SQL Server, por questões de limitação do Driver do Firedac no Delphi Comunnity Edition.
